Versionamento

O git é um software que permite realizar versões de um projeto.O gitHub desktop é um software que faz a ligação entre Visual Studio Code , Git e GitHub.

O GitHub desktop permite fazer a sincronia entre o repositório local e o repositório remoto

Commit

É o processo de confirmação de uma versão, ou seja, a cada alteração é possível confirmar a alteração

Push

Este comando envia(sincroniza) as alterações realizadas no repositório local com o repositório remoto no GitHub
